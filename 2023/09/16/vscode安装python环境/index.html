vscode配置python环境

发表于 2023-09-16 | 更新于: 2023-09-16 | 分类于 python |
字数统计: 780 字 | 阅读时长 ≈ 2 分钟

1 下载并安装vscode

1.1 下载地址

https://code.visualstudio.com/Download

1.2 安装vscode（其他系统安装方式这里不再赘述，这里推荐使用.zip方式安装）

  • 选择对应机器以及版本下载zip包
  • 解压之后在和Code.exe同级的目录下创建data文件夹(如果没有按照这步来，vscode可能就会在一些你无法预知的目录输出数据，比如在系统盘下的Appdata或program files目录下)
  • 这样相关插件&用户数据就都会在data文件夹下(这也是推荐使用.zip安装vscode的理由)

2 下载python解释器

2.1 下载地址(官网有个时候打不开，多刷几次)

https://www.python.org/downloads/

2.2 安装python解释器(以python-3.10.0-amd64.exe为例)

  • 双击python-3.10.0-amd64.exe进入安装界面
  • 务必勾选最下面Add Python 3.10 to PATH（这里是自动帮你将python路径加到环境变量中去，如果这里不勾选，后面需要自己手动去添加python环境变量：计算机->属性->更改配置->高级->环境变量->将python安装路径以及python下的Scripts路径加到用户变量的path中或者系统变量的PATH中)
  • 选择customize installation
  • 建议：Optional Features使用默认勾选 -> Advanced Options除了download debug相关不勾选，其他都勾选，安装路径自定义选择
  • 点击安装

3 vscode配置python环境

vscode配置python环境主要就是插件安装以及vscode用户配置

3.1 插件安装（注意插件全部安装完之后重启vscode）

  • 搜索插件”中文”然后选择安装(汉化vscode)
  • 搜索插件”Python”并点击安装(配置vscode python环境最主要的插件)
  • 搜索插件”intellicode”并点击安装(代码提示&补全)
  • 搜索插件”pylance”并点击安装(intellicode插件针对python只支持pylance语言服务器)
  • 搜索插件”AREPL”并点击安装(实时执行python代码，类似预览效果，非必须安装)

3.2 用户配置

  • 检查python环境是否生效：新建一个.py文件，正常是可以在vscode左下角看到安装的python版本，并且点击左下角可以切换python版本
  • 代码提示与自动补全配置：选择：文件->首选项->设置，然后搜索”language server”，选择出现的Python那一栏，最下面那个，如果是”Jedi”或者其他服务器，则需要替换成”Pylance”，否则没有代码提示和自动补全
  • 指定默认python版本：如果有多个python解释器版本，并且想要设置默认的python解释器，那么可以在用户的”settings.json”文件中添加需要指定的python版本路径：比如："python.pythonPath": "D:\\Pythonpkg\\python3.11.5"
  • 静态检查配置：vscode配置完python环境后，右下角可能会报找不到pylint或者报pylint不存在，这主要是python静态检查需要用到的配置，注意这里要选择prospector进行安装(耗时可能会有点久)，不要选择其他的
